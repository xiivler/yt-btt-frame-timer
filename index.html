YouTube BTT Frame Timer

About

Finds the time of a YouTube segment in seconds and 60 FPS frames. Source code is available. Based on slashinfinity's yt-frame-timer.

Start and End Times

Find the start frame (you can use the , and . keys to find the exact frame). Right click the video and select “Copy debug info” and paste it for starting frame. Now, navigate to the end frame only using the ,, .., left, and right keys (do not press play, click on the time bar, or use the numbers 1-9).

Tip: if you put the video at 0.25 speed, the left and right arrows will skip forward and back only 1.25 seconds.
